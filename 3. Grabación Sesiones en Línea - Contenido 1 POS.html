C O N C L U S I O N E S
Los sistemas de gestión de bases de datos son utilidades para la definición y gestión de las estructuras de archivo para la manipulación y gestión en general de datos estructurados, mientras que el lenguaje SQL corresponde al lenguaje universal, en lo que se refiere al SQL ANSI, para la interacción con las bases de datos implementadas mediante los sistemas manejadores de bases de datos, teniendo en cuenta que independientemente al sistema manejador de bases de datos que se emplee en un proyecto, siempre se va a emplear el lenguaje SQL en conjunto con el lenguaje de programación empleado para la implementación de los sistemas de información.
